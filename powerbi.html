Introduction to POWER BI

Microsoft Power BI is an interactive data visualization software product developed by Microsoft with a primary focus on business intelligence.[2] It is part of the Microsoft Power Platform. Power BI is a collection of software services, apps, and connectors that work together to turn various sources of data into static and interactive data visualizations.
